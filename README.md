# Astraea

Interview preparation platform for object-oriented design and API-based coding interviews.

## Stack

- Next.js App Router + TypeScript
- TailwindCSS
- PostgreSQL + Prisma
- Next.js API routes

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create env file:
   ```bash
   cp .env.example .env
   ```
3. Generate Prisma client:
   ```bash
   npm run prisma:generate
   ```
4. Run dev server:
   ```bash
   npm run dev
   ```
